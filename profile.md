# **Perfil de Orador Profesional y Portafolio de Ingeniería: Cristian Moreno Zuluaga**

## **Análisis Ejecutivo y Trayectoria Profesional**

Cristian Moreno Zuluaga, conocido ampliamente en la comunidad global de tecnología como "Khriztian Moreno", es un Ingeniero de Inteligencia Artificial (IA) Fullstack, evangelista técnico y líder comunitario con más de 15 años de trayectoria en el diseño, desarrollo e implementación de arquitecturas web de alta escalabilidad1. Con base en Medellín, Colombia, su experiencia profesional se ha consolidado a través de su desempeño técnico en compañías líderes de la industria como MercadoLibre y Globant, además de consultoras especializadas como Mattersupply4. A lo largo de su carrera, se ha destacado por un balance excepcional entre la ejecución técnica rigurosa y un profundo compromiso con la transferencia del conocimiento en entornos de código abierto2.
La filosofía de Moreno Zuluaga se fundamenta en el principio pedagógico de "aprender haciendo" (_learn by doing_), utilizándola como base para crear contenido técnico de alta calidad en forma de artículos especializados, cursos estructurados y videos formativos2. Este enfoque pragmático le ha permitido colaborar de manera directa con bootcamps de desarrollo de software acelerado como Make It Real, donde se ha desempeñado como mentor principal diseñando guías minuciosas para la configuración inicial y personalizada de proyectos basados en React sin depender de abstracciones preconfiguradas4.
Su capacidad para inspirar y educar a grandes audiencias se consolidó de manera histórica al formar parte activa como docente principal en el hito que obtuvo el Guinness World Record por dictar la clase de programación presencial más grande del mundo en Plaza Mayor (Medellín, Colombia), logrando reunir a más de 3,119 estudiantes para aprender desarrollo de software y tecnologías cloud utilizando HTML, CSS y JavaScript. En el ámbito corporativo, su desempeño técnico se complementa con el dominio de metodologías ágiles como Scrum para optimizar los ciclos de lanzamiento de productos de software y alinear los objetivos del negocio con las capacidades de los equipos de ingeniería8.

## **Creación de Contenido Técnico de Vanguardia y Publicaciones**

A través de sus blogs oficiales, canales interactivos y plataformas como Medium y dev.to, Cristian Moreno genera recursos formativos para desarrolladores con un enfoque riguroso de ingeniería de software. Sus publicaciones destacan por analizar temas complejos de rendimiento, optimización y herramientas modernas en tendencia2:

### **1\. Inteligencia Artificial, Agentes y el Futuro de la Web (2024-2026)**

- **AI-Powered Web Debugging \- From Browser Runtime to Autonomous Agent (May 2026):** Publicación técnica que explora el uso del Model Context Protocol (MCP) para conectar agentes de IA al motor de renderizado de Chrome, permitiendo la depuración autónoma en tiempo real de fugas de memoria, cuellos de botella de rendimiento y QA automatizado.
- **The 15 Chrome Updates from Google I/O 2026 We Actually Need to Care About (May 2026):** Guía práctica de novedades del ecosistema de Google I/O, destacando WebMCP, DevTools impulsados por IA, Gemini integrado nativamente en el cliente y APIs de navegación suave (Soft Navigations).
- **The Agent Development Kit \- ADK (Oct 2025):** Análisis arquitectónico del framework ADK para simplificar la creación de agentes autónomos inteligentes, detallando sus seis componentes clave: agente, ejecutor (runner), sesión, estado, memoria y herramientas personalizadas con MCP.
- **The revolutionary Chrome API (Aug 2025):** Introducción de corte técnico para el uso práctico de Prompt API, permitiendo a los desarrolladores web ejecutar modelos de lenguaje integrados en el navegador (como Gemini Nano) sin latencia del servidor ni costes de infraestructura de APIs tradicionales.
- **Desbloquea tu creatividad con Google Gemini y JavaScript: Guía práctica (Mar 2024):** Tutorial exhaustivo paso a paso para la integración fluida de modelos generativos avanzados de IA de Google de manera local empleando JavaScript nativo.

### **2\. Rendimiento Web y Optimización Profesional de Interfaces (2025-2026)**

- **Mastering Chrome DevTools for Web Performance Optimization (Nov 2025):** Guía avanzada para ingenieros frontend sénior sobre cómo diagnosticar rigurosamente métricas complejas de Core Web Vitals, concentrándose en la detección de retrasos en Interaction to Next Paint (INP), Largest Contentful Paint (LCP) y Cumulative Layout Shift (CLS).
- **Demystifying Core Web Vitals & Optimizing for Speed (Oct-Nov 2025):** Serie de guías detalladas para analizar datos de campo reales de usuarios (CrUX) y optimizar la carga a través del hilo principal, estabilidad visual de maquetado y optimización de recursos multimedia.
- **Navigating the Future \- Soft Navigations for SPAs (Nov 2025):** Explicación del funcionamiento de la nueva API de Soft Navigations de Chrome orientada a estandarizar y reportar correctamente las mediciones de rendimiento en el ciclo de vida de aplicaciones de una sola página.
- **Beyond the Basics \- Advanced CSS Techniques for Web Developers (Sep 2025):** Análisis de rendimiento del motor de renderizado de CSS de los navegadores modernos, discutiendo el coste de propiedades específicas, el uso eficiente de Container Queries, Subgrid, pseudo-clases dinámicas (:has()) y propiedades personalizadas para diseño a escala.
- **Web Components and Material Design (Sep 2025):** Estrategias de encapsulamiento del Shadow DOM, elementos personalizados y tokens de diseño avanzados con Lit para crear microfrontends interoperables y escalables.

### **3\. Arquitectura de Software y Ecosistema Moderno de JS (2024-2025)**

- **The Web Stack You Cannot Ignore in 2026 (Dec 2025):** Análisis de tendencias dominadas por APIs nativas de Fugu/PWA, autenticación sin contraseñas (Passkeys / FedCM), APIs de IA integradas y arquitecturas que minimizan el excesivo envío de JavaScript innecesario al cliente.
- **What is a 12 Factor App \- a Quick Guide (Jan 2025):** Manual pragmático de los doce principios metodológicos para el diseño de aplicaciones modernas de software como servicio (SaaS) altamente escalables, portables y listas para integraciones Cloud continuas.
- **Tech Stack 2025 (Jan 2025):** Evaluación de balances de ingeniería para decidir el stack ideal de desarrollo web fullstack rápido y mantenible en la escena actual de producción.
- **¡Rust está revolucionando el desarrollo JavaScript\! (Sep 2024):** Explicación técnica de Rspack v1.0, el empaquetador veloz escrito en Rust, compatible con el ecosistema de plugins de webpack y diseñado para acelerar de forma drástica los tiempos de compilación de aplicaciones monolíticas en entornos empresariales.
- **Server Components Vs. Server-side Rendering (Jan 2024):** Discusión analítica sobre la gestión de estado y el balance de renderizado entre cliente y servidor en arquitecturas modernas de React.
- **WebContainers are having a moment (Oct 2024):** Reseña e implicaciones técnicas del lanzamiento de bolt.new de StackBlitz, un sandbox de desarrollo potenciado por IA que corre de manera nativa Node.js en el cliente gracias a WebAssembly.

### **4\. Producción Multimedia y Series de Videotutoriales**

- **Curso Básico de React.js (YouTube \- 2024):** Serie educativa completa que aborda de manera clara la filosofía de React, el uso de Vite, paso de Props, manejo avanzado del estado local con el hook useState, ruteo con React Router, y side-effects con el ciclo de vida de componentes mediante useEffect.
